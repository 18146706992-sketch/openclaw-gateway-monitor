# OpenClaw Gateway Monitor\n\nOpenClaw Gateway 自动化监控脚本，支持自动重启和故障排除。\n\n## 功能特性\n\n- **开机自启动**：通过 Windows 计划任务实现开机自动运行\n- **健康监控**：使用 openclaw health 和 gateway status 命令检测状态\n- **自动修复**：重启失败时自动重试（默认 3 次，间隔 60 秒）\n- **故障排除**：修复失败时自动执行诊断并生成报告\n- **日志记录**：日志保存到 D:\.openclaw\logs\\n\n## 文件说明\n\n| 文件 | 说明 |\n|------|------|\n| openclaw-monitor.ps1 | 核心监控脚本 |\n| install-openclaw-monitor.ps1 | 安装/卸载脚本 |\n\n## 快速开始\n\n### 安装（自动开机自启动）\n`powershell\npowershell -ExecutionPolicy Bypass -File install-openclaw-monitor.ps1 -RunNow\n`\n\n### 常用命令\n\n`powershell\n# 单次检测模式\npowershell -ExecutionPolicy Bypass -File openclaw-monitor.ps1 -Once\n\n# 持续监控模式（默认 60 秒检测一次）\npowershell -ExecutionPolicy Bypass -File openclaw-monitor.ps1\n\n# 自定义参数\npowershell -ExecutionPolicy Bypass -File openclaw-monitor.ps1 -CheckInterval 30 -MaxRetries 5 -RetryWait 120\n`\n\n### 卸载\n`powershell\npowershell -ExecutionPolicy Bypass -File install-openclaw-monitor.ps1 -Uninstall\n`\n\n## 参数说明\n\n| 参数 | 默认值 | 说明 |\n|------|--------|------|\n| -CheckInterval | 60 | 检测间隔（秒）|\n| -MaxRetries | 3 | 最大重启重试次数 |\n| -RetryWait | 60 | 重试间隔（秒）|\n| -Once | false | 单次检测后退出 |\n| -LogDir | D:\.openclaw\logs | 日志目录 |\n\n## 工作流程\n\n`\n检测 Gateway 状态\n       |\n   正常 → 不做任何操作\n       |\n   异常 → 启动/重启 Gateway\n       |\n   等待 60 秒 → 再次检测\n       |\n   依然异常 → 重试（最多 3 次）\n       |\n   全部失败 → 执行故障排除\n       |\n   生成诊断报告 → 保存到 diagnostics\\n`\n\n## 日志文件\n\n- D:\.openclaw\logs\openclaw-monitor-YYYYMMDD.log — 每日监控日志\n- D:\.openclaw\logs\diagnostics\diagnostic-YYYYMMDD-HHmmss.txt — 诊断报告\n\n## 常用管理命令\n\n`powershell\n# 查看计划任务状态\nGet-ScheduledTask -TaskName 'OpenClawGatewayMonitor' | Get-ScheduledTaskInfo\n\n# 查看实时日志\nGet-Content 'D:\.openclaw\logs\openclaw-monitor-20260415.log' -Tail 20 -Wait\n\n# 停止监控\nStop-ScheduledTask -TaskName 'OpenClawGatewayMonitor'\n\n# 手动启动监控\nStart-ScheduledTask -TaskName 'OpenClawGatewayMonitor'\n`\n\n## 健康判断标准\n\n满足以下任一条件即视为健康：\n- openclaw health 输出包含 Feishu: ok\n- openclaw gateway status 输出 Runtime: running\n- openclaw gateway status 输出 RPC probe: ok\n\n## 故障排除\n\n当脚本无法自动修复时，会执行以下诊断命令并生成报告：\n1. openclaw status\n2. openclaw gateway status\n3. openclaw doctor\n4. openclaw channels status --probe\n\n诊断报告保存在 D:\.openclaw\logs\diagnostics\ 目录。\n\n官方故障排除文档：https://docs.openclaw.ai/troubleshooting\n
